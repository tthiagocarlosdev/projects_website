SNAKE_CASE

Este projeto recebe um texto qualquer e retorna uma string no formado snake_case.

Também informa ao final, o tamanho da string, ou seja, a sua quantidade de caracteres.

Criei esssa aplicação, para facilitar na hora de colocar os nomes de algumas aulas nos arquivos do GitHub.
